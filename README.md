# Single Page Web Application

This is a basic single page web application template with separate HTML, CSS, and JavaScript files.

## Project Structure

```
├── src/
│   ├── index.html
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── main.js
└── public/
    └── (for static assets)
```

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

This will start a live server and open your application in the default browser.

## Development

- Edit `src/index.html` for your HTML structure
- Add your styles in `src/css/styles.css`
- Write your JavaScript code in `src/js/main.js`
- Place any static assets (images, fonts, etc.) in the `public` directory
